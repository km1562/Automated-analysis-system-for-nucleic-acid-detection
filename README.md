# Automated-analysis-system-for-nucleic-acid-detection
三码（核酸检测、健康码和行程码）的审核分析，是有效发现并且阻断疫情传播的方式之一，很多机关单位、政府部门都需要检查三码或者其中之一部分，才能让相关的人员进出场所，然而三码的审核分析却需要耗费核查人员的大量时间。所以实现三码得关键信息提取和分析自动化系统十分重要，能够显著提高审核效率
